# 常用命令

## 查询端口号占用

```shell
lsof -i tcp:<端口号>
```

## 查询当前目录下磁盘使用情况

```shell
du -h -d <目录深度>
```

## 查询当前磁盘占用情况

```shell
df -h
```

## 查询当前运行内存占用情况

```shell
# total：系统总的物理内存大小
# used：已使用的物理内存大小
# free：空闲的物理内存大小
# shared：被共享的内存大小
# buff/cache：被缓存的内存大小
# available：可用的内存大小，即可供程序使用的内存大小
free -h
```
