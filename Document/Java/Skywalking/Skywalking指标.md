# Skywalking指标

| 指标                                         | 作用                                  |
| -------------------------------------------- | ------------------------------------- |
| Service Apdex                                | 服务Apdex评分                         |
| Success Rate                                 | 请求成功率，单位:百分比               |
| Service Load(calls/min)                      | 每分钟请求数量                        |
| Service Avg Response Time(ms)                | 平均响应延时，单位:毫秒               |
| Service Response Time Percentile(ms)         | 百分比响应延时，单位:毫秒             |
| Servce Instances Load(calls/min)             | 每个服务实例的每分钟请求数量          |
| Show Service Instance                        | 每个服务实例的最大延时                |
| Service Instance Successful Rate             | 每个服务实例的请求成功率，单位:百分比 |
| Endpoint Load in Current Service             | 每个端点的每分钟请求数量              |
| Slow Endpoints in Current Service            | 每个端点的最慢请求时间，单位:毫秒     |
| Endpoints Successful Rate in Current Service | 每个端点的请求成功率，单位:百分比     |
