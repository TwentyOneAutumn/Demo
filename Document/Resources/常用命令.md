# 常用命令

| Command                          | 说明                       | 环境              |
| -------------------------------- | -------------------------- | ----------------- |
| netstat -ano \| findstr <端口号> | 查询端口号占用             | `Linux` `Windows` |
| du -h -d <目录深度>              | 查询当前目录下磁盘使用情况 | `Linux` `Mac`     |
| df -h                            | 查询当前磁盘占用情况       | `Linux`           |
